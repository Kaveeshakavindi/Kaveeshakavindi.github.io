React Router
#React
After creating the 'Home' component and the 'About' component, we can create a component that defines our application's routes using 'BrowserRouter' and 'Route' from the 'react-router-dom' package.
App.js
function App(){
return (
<BrowserRouter>
<Route exact path = "/"component={Home}/>
<Route exact path = "/about"component={About}/>
</BrowserRouter>
);
}
export default App;
Finally, you can render the App component in your main index.js file:
index.js
ReactDOM.render(
<React.StrictMode>
<App/>
</React.StrictMode>,
document.getElementById('root')
);

grid-template-columns
#CSS
To make the first column smaller and the second column bigger, we can specify the width of each column using the fr unit, which represents a fraction of the available space. Here's an example:
file.css
.container{
display: grid;
grid-template-columns: 1fr 2fr;
}
Here, first column is 1/3 of available space, second column is 2/3.
